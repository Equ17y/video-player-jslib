# Видеоплеер

**Демо:** [https://equ17y.github.io/video-player-jslib/](https://equ17y.github.io/video-player-jslib/)

## Функции
- Воспроизведение/пауза видео
- Громкость вкл/выкл
- Полноэкранный режим
- Тень вокруг плеера
- Отображение текущего времени и длительности видео

### Анимации
- Плавное появление/исчезновение ползунка при наведении
- Анимация прогресс-бара

## Установка
1. Клонировать репозиторий
```
   git clone https://github.com/Equ17y/video-player-jslib.git
```   

2. Открыть `index.html` в браузере
